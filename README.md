# The Enhancer - Photo - Video

## Purpose
This repository contains automation workflows for photo and video editing, exports, client delivery, and related media processes.

## Main folders
- src/file_ingest = incoming files and intake workflows
- src/editing_workflows = editing automations
- src/exports = export/render workflows
- src/client_delivery = delivery processes
- src/social_media = social posting workflows
- docs = business workflows and planning
- templates = reusable templates
- assets = working assets
- tests = testing code

## Setup
Create a local .env file from .env.example and add your real keys.

## Notes
Do not upload private client data or real API keys.
